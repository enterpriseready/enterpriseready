Featured Post: Product Assortment
Product assortment is really more than a feature for EnterpriseReady. It is the foundation on which the concept was initially conceived. The other 10 features were shaped by studying how the most successful SaaS companies use features to differentiate their pricing plans (therefore creating product assortment.) An understanding of this core concept is the key to capturing the value that the EnterpriseReady features can generate.

Wanted: Contributors
One of the goals of EnterpriseReady is to create a common vernacular for building enterprise software features. We open-sourced the entire project so anyone can create their own version or contribute edits back by simply sending us a pull request (special thanks to recent contributions from Segment and Michael Dearing). We welcome feedback, ideas and suggestions so please drop us a line whenever (just reply to this email). If you are looking for inspiration on how to contribute, we just created an initial set of help wanted issues on Github.
